Học HTML: Bước đầu để xây dựng website

1. Giới thiệu

Khi bắt đầu học lập trình web, HTML là kiến thức đầu tiên mà người học cần nắm. HTML giúp tạo cấu trúc nội dung cho một trang web.

HTML không phải là ngôn ngữ lập trình xử lý thuật toán, mà là ngôn ngữ đánh dấu. Nó giúp trình duyệt hiểu đâu là tiêu đề, đoạn văn, hình ảnh, liên kết và form.

Với mình, học HTML là bước khởi đầu quan trọng để học tiếp CSS, JavaScript và các công nghệ web khác.

2. Vì sao nên học HTML?

HTML giúp người học hiểu cách một website được tổ chức. Ví dụ, ta có thể nhận biết phần header, main và footer.

Một lợi ích khác của HTML là cú pháp dễ học. Chỉ cần hiểu cách mở thẻ, đóng thẻ và lồng thẻ hợp lý, ta đã có thể tạo một trang web đơn giản.

Tuy nhiên, để viết HTML tốt, người học cần chú ý đến semantic HTML. Đây là cách sử dụng đúng thẻ theo ý nghĩa của nội dung.

"Code is like humor. When you have to explain it, it’s bad."

- Cory House

3. Cấu trúc cơ bản của HTML

Một file HTML thường bắt đầu bằng khai báo <!DOCTYPE html>. Sau đó là thẻ html, bên trong gồm head và body.

Phần head chứa thông tin cấu hình như bảng mã, tiêu đề và link CSS. Phần body chứa nội dung hiển thị trên trình duyệt.

Khi viết code, mình thường dùng Ctrl + S để lưu file. Nếu chạy đúng, trình duyệt có thể hiển thị kết quả như: Trang web đã được tải thành công.

Một số ký hiệu như H2O hoặc HTML5 có thể được trình bày bằng thẻ sub và sup.

4. Ví dụ code

Ví dụ 1: Cấu trúc HTML cơ bản

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang HTML đầu tiên</title>
</head>
<body>
    <!-- Tiêu đề chính của trang -->
    <h1>Xin chào HTML</h1>

    <!-- Đoạn mô tả ngắn -->
    <p>Đây là trang web đầu tiên của tôi.</p>
</body>
</html>

Ví dụ 2: Tạo liên kết và hình ảnh

<!-- Link mở trong tab mới -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
    Truy cập website
</a>

<!-- Hình ảnh có alt text -->
<img src="images/avatar.png" alt="Ảnh đại diện cá nhân" width="200" height="200">
